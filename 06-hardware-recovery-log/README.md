This project documents hands-on hardware recovery and troubleshooting activities performed as part of my homelab development.

The goal is to improve hardware diagnostics skills and recover usable components from discarded systems.

## Activities Documented

- Hardware diagnostics
- Component testing
- System rebuilding
- Soldering practice
- Workspace organization

## Recovered Systems

### Street PC

A discarded PC was recovered and disassembled.

Goals:

- Identify working components
- Analyze hardware failure points
- Salvage reusable parts

---

### Dead PC Diagnostics

Another non-functional system was disassembled to practice troubleshooting techniques.

Focus areas:

- Identifying failed components
- Testing power supplies
- Verifying motherboard functionality

## Motherboard Repair Attempt

A motherboard repair attempt was performed involving soldering.

Purpose:

The repair attempt was unsuccessful but provided valuable learning experience.
