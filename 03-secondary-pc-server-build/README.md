# Secondary PC Server Build

This project documents the build of a custom secondary PC used as both a homelab server and a secondary gaming system for friends.

The system replaces an older HP Pavilion that previously served this role.

---

## Purpose

The machine serves two main roles:

1. Home server for experimentation and services
2. Secondary gaming PC for guests and LAN-style sessions

This dual-purpose design maximizes the use of available hardware.

---

## Hardware Specifications

CPU  
AMD Ryzen 7 2700X

Motherboard  
B450 Micro-ATX motherboard (Z-Sus Chinese brand)

Memory  
16GB DDR4 running at 2400 MHz

GPU  
PowerColor Radeon RX 470

Storage  

256GB NVMe SSD (Blu-ray brand)  
512GB SATA SSD (Sonambulus brand)

Power Supply  

MSI 650W PSU

Cooling  

Air cooler (€20 purchase)

Case  

Micro-ATX case
