# HP EliteDesk vs AMD A10 System

This project documents the comparison between two systems used during my early homelab experimentation.

The goal was to determine which system would perform better for basic server workloads and experimentation.

---

## Systems Compared

### HP EliteDesk

Typical configuration:

CPU: Intel Core i5 (varies by model)  
RAM: 8GB DDR3  
Storage: SATA SSD or HDD  
Form Factor: Small Form Factor (SFF)

Advantages:

- Business-grade hardware
- Reliable build quality
- Low power consumption
- Compact size

Limitations:

- Limited GPU capability
- Limited upgrade flexibility

---

### AMD A10 System

Configuration:

CPU: AMD A10 7800  
RAM: 8GB DDR3  
Storage: HDD or SSD

Advantages:

- Strong integrated graphics
- More flexible motherboard upgrades
- Good for light desktop workloads

Limitations:

- Higher power consumption
- Older architecture

---

## Testing Goals

The systems were evaluated for:

- Linux compatibility
- General responsiveness
- Potential server usage
- Experimentation environment

---

## Outcome

Both systems provided useful learning opportunities.

However, the experience eventually led to the creation of a more capable custom-built secondary PC using:

Ryzen 7 2700X  
16GB DDR4  
RX 470  

This marked the transition toward building a more serious homelab environment.

---

## Lessons Learned

- Older hardware can still be useful for learning
- Linux runs well on modest systems
- Hardware experimentation helps build troubleshooting skills
